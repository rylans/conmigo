# conmigo
Task manager and time tracker

## Quick Start

#### Download and install

  go get github.com/rylans/conmigo

## Usage

#### Start a new task

  conmigo -start "Buy some eggs"

#### List tasks

  conmigo
  
#### Search tasks with regular expressions

  conmigo | grep "keyword"

#### Count open tasks

  conmigo | wc -l
